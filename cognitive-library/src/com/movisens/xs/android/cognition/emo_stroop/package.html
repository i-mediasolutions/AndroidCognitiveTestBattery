Provides the classes for the execution of the Emotional Stroop Test.

The Emotional Stroop Test -Activity starts with an instruction screen that displays how the test works. After pressing the start button the test begins displaying the first word on the screen that has to be assessed. The lower screen displays buttons by what a color can be chosen for the actual displayed word. After the assessment of all words the test ends and the results can be viewed.
